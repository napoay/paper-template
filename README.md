# paper-template

A latex paper template that supports both Chinese and English.

## 1. usage

There are two files in the tex folder: paper.tex and ref.bib, which are tex source code and references.

compile steps:

```
xelatex paper.tex

bibtex paper.aux

xelatex paper.tex
```

Finally, paper.pdf is generated.
